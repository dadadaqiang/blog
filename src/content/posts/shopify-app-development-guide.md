## 一、Shopify 是要钱的吗？

最简单直接的答案：**是，Shopify 是一个付费的 SaaS 建站平台。**

### 月费方案

| 方案 | 月付价 | 年付价 | 适合 |
|------|--------|--------|------|
| Starter | $5/月 | — | 社交媒体销售 |
| Basic | $39/月 | $29/月 | 个人创业者起步 |
| Grow（原 Shopify） | $105/月 | $79/月 | 小型团队 |
| Advanced | $399/月 | $299/月 | 中大型业务 |
| Plus | 起 $2,300/月 | — | 大型企业 |

### 还有额外费用

- **交易手续费**：每笔订单 2.0%~2.9% + 30¢（用 Shopify Payments 可省掉第三方手续费）
- **域名**：年费约 $10~$15
- **App/插件订阅**：很多功能需要额外付费
- **付费主题**：$100~$350

### 有没有免费？

**没有永久免费计划**，但提供：

- **3 天免费试用**（无需绑定信用卡）
- 续费后 **前 3 个月 $1/月** 优惠

> 如果只是想开个店做生意，最低 $5/月（Starter）起步，完整独立站最低 $29/月（Basic 年付）。

## 二、开发 Shopify 插件要钱吗？

重点来了：**开发插件本身不需要付费**。

### 开发阶段：完全免费

| 项目 | 费用 |
|------|------|
| 注册 Partner 账号 | **免费** |
| 创建 Development Store（测试商店） | **免费**（无限个） |
| 使用 Shopify CLI / API | **免费** |
| 给自己或客户做私有插件（Custom App） | **免费** |

开发商店（Development Store）功能相当于 **$299/月的 Advanced 套餐**，而且**没有时间限制**，内置 Bogus Gateway 可以无限模拟交易测试。

### 发布到 App Store：一次性 $19

如果想把插件上架到官方 App Store 让所有人下载，需要支付 **一次性 $19 账号注册费**（2025 年价格）。之后就**不用再交月费**。

### 收入分成

| 收入区间 | 平台抽成 |
|---------|---------|
| 前 $1,000,000 | **0%**（全部归你） |
| 超过 $100 万 | **15%** |
| 支付手续费 | **2.9%** |

## 三、怎么测试 Shopify 插件？

完整的测试流程分几步走，全部免费：

### 1. 准备开发环境

```bash
npm install -g @shopify/cli@latest
shopify app init my-app --template react
cd my-app
```

### 2. 创建开发商店

在 [Shopify Partners 后台](https://partners.shopify.com) → **Stores → Add store → Development store**，几分钟建好一个免费的测试商店。

### 3. 本地运行并测试

```bash
shopify app dev
```

这行命令会自动：

- **创建 Cloudflare Tunnel**，把本地开发服务器暴露到公网
- **打开浏览器**让你选择开发商店安装插件
- **监听文件变化**，代码改完自动热更新
- 直接在开发商店后台实时查看插件效果

### 4. 模拟交易测试

开发商店内置 **Bogus Gateway**（虚拟支付网关）：

```
信用卡号: 1   →  模拟成功
信用卡号: 2   →  模拟失败
```

可以无限次测试订单流程，不收一分钱。

## 四、注册 Partner 时 "Business Located" 怎么选？

注册 [Shopify Partners](https://partners.shopify.com) 时有一个 **Business Located** 字段，这个选了就不能随便改，而且确实有实际影响。

### 不同地区决定了什么？

| 地区 | 签约方 |
|------|--------|
| 中国大陆 | Shopify Commerce Singapore Pte. Ltd.（新加坡） |
| 香港 | Shopify Commerce Singapore Pte. Ltd. |
| 美国 | Shopify Inc.（加拿大） |
| 欧洲 | Shopify International Limited（爱尔兰） |

### 对插件开发者的实际影响

| 事项 | 选中国大陆 | 选香港 | 选美国 |
|------|-----------|--------|-------|
| 注册免费 | ✅ | ✅ | ✅ |
| 开发测试 | ✅ | ✅ | ✅ |
| 上架 App Store（$19） | ✅ | ✅ | ✅ |
| 收入提现 | ✅ Payoneer/电汇 | ✅ 银行转账 | ✅ 多种方式 |
| 税务 | W-8BEN（免预扣税） | W-8BEN | W-9（需美国税号） |

**不支持的国家/地区**：古巴、伊朗、朝鲜、叙利亚、克里米亚等（完整列表见 Shopify 官方文档）。

### 我的建议

- **人在大陆、没有香港账户** → 选 **China**，完全没问题
- **有香港公司或银行账户** → 选 **Hong Kong SAR**，提现更方便
- **不要为了方便乱选美国** → 美国要求提供 SSN/ITIN（美国税号），你没有的话反而麻烦

> 插件开发和测试阶段不受地区选择影响——开发商店、API、CLI 功能全球统一。

## 五、提现方案深度对比

这是最容易被忽略但实际损失最大的环节。同样赚 $1,000 USD，不同提现方案到手差很多。

### 方案一：选 China → 电汇到内地银行卡

| 环节 | 费用 |
|------|------|
| Hyperwallet 转出 | 免费 |
| SWIFT 中间行手续费 | $20~$30 USD |
| 内地银行入账费 | $10~$15 USD |
| **总损失** | **约 $30~$45（3%~4.5%）** |

### 方案二：选 Hong Kong SAR → 香港银行账户

| 环节 | 费用 |
|------|------|
| Hyperwallet → 香港本地转账（FPS/CHATS） | **免费** |
| 中间行费用 | **$0**（本地转账不走 SWIFT） |
| **如果能保留美元**（多币种账户） | **损失几乎为 0** |
| 如需换港币 | 约 **0.5%~1%** |

### 方案三：选 China → PayPal → 内地银行卡

| 环节 | 费用 |
|------|------|
| PayPal 提现手续费 | **$35 USD（固定）** |
| PayPal 汇率加价 | **2.5%~3%**（隐藏汇率差） |
| **总损失** | **约 $55~$65（5.5%~6.5%）** ❌ |

### 实际到手对比

| 方案 | $1,000 USD 到手 | 损失 |
|-----|----------------|------|
| 🥇 **香港银行账户（留美元）** | **$1,000** | **$0** |
| 🥇 **香港银行账户（换港币）** | **≈ $990** | **$10（1%）** |
| 🥈 **内地银行电汇** | **≈ $965** | **$35（3.5%）** |
| ❌ **PayPal 到内地卡** | **≈ $940** | **$60（6%）** |

### 为什么 PayPal 最亏？

PayPal 的收费模式是"两头吃"：

1. **提现手续费**：每笔 $35 USD（固定，不论金额大小）
2. **汇率加价**：PayPal 的美元兑人民币汇率比市场中间价差 **0.15~0.20**，相当于多收了 **2.5%~3%**

以 $1,000 USD 为例：
- 市场汇率（7.25）→ ¥7,250
- PayPal 汇率（约 7.10）→ ¥7,100
- **汇率差额 = ¥150 凭空蒸发**
- 加上 $35 提现费（≈¥250）
- **总共损失 ≈ ¥400，相当于 5.5%**

### 为什么香港账户优势这么大？

1. **本地转账免费**：Hyperwallet → 香港银行走 FPS/CHATS，不走 SWIFT，没有中间行扣费
2. **可以留美元**：香港银行普遍支持多币种账户，USD 直接存着不用换
3. **想换人民币也不亏**：通过 Wise 等渠道换汇，汇率远好于 PayPal
4. **Shopify Payments 也能用**：如果以后自己做商家，香港账户还能用 Shopify 官方收款

1. 注册 Shopify Partners（选 **China**）
2. 创建开发商店
