这个五一假期，又折腾了一遍。

信用卡过期了，激活信用卡的时候发现之前用的contabo服务器涨价了。

$4.99涨价到$7.2，囊中羞涩，加上之前contabo一直速度不佳，于是决定换服务商了。

想要搭一个富强VPS自用，要做的事情如下：

*   选一个离自己地理位置近的

美国西海岸：洛杉矶、圣何塞、西雅图

东亚地区：日本、新加坡

*   看延迟、丢包

一般vps提供商都会给出测试vps的ip，或者looking glass给大家测试。

直接ping一下看看。

```
ping XX.XX.XX.XX -t
```

延迟300ms以下还行，丢包最好不丢，要丢也是20%以下差不多。

*   看线路

下载ipip.net的BestTrace

```
https://ipip.net/product/client.html
```

测一下自己本地到VPS的线路是走哪里的。

```
中国电信：
中国电信的 163 网络，一般会经过市级 → 省级 → 国际出口 → 境外接入点 AS（自治路由协议） 号为 4134 的路由节点，这些路由节点的 IP 地址开头一律是 "202.97.*.*"，全程不会经过 CN2 网络节点；
CN2 GIA 产品在从市级 → 省级 → 国际出口 → 境外接入点的过程中，全程走 AS 号为 4809 的路由节点，这些路由节点的 IP 地址开头一律是 "59.43.*.*"，全程不会经过 163 网络节点；

中国移动：
中国移动的进出口跳跃性很大，国内部分必走 AS9808 路由（221.176.*.*/221.183.*.*）

中国联通：
中国联通普通用户从市级 → 省级 → 国际出口 → 境外接入点的过程中，全程走 AS 号为 4837 的路由节点，这些路由节点的 IP 地址开头一律是 "219.158.*.*"，全程不会经过 A网（AS9929）
```

*   看下载速度

从VPS下载一个文件，其实就是测vps的上传速度。

因此，直接从looking glass上下载一个100M/1000M的文件，看看下载速度如何。

*   价格和优惠码

当然，买VPS还是得看价格。

买什么都有优惠和折扣，所以下单之前先找找有没有VPS的优惠折扣。
